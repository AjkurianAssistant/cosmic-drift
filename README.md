# Cosmic Drift 🌌

An interactive particle universe built with vanilla HTML5 Canvas.

## Features
- 600 glowing particles with nebula blobs and connection lines
- Three interaction modes: **Attract**, **Explode**, **Orbit**
- Touch & mouse support — optimized for mobile
- Cycles through colour palettes on each mode switch
- Zero dependencies — single HTML file

## Usage
Just open `index.html` in any browser. Touch/drag to shape the cosmos.
